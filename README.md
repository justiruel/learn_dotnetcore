# Perintah dasar

## Help
```
dotnet --help
```

## Create new API project
``` 
dotnet new webapi -o TodoApi
```

## Build Project
```
dotnet build
```

## Jalankan project
```
dotnet run
``` 

## dotnet restore (sama dengan npm install, dipergunakan jika library sudah tercatat di .csproj)
```
dotnet restore
```

## publish
```
dotnet publish -c Release -o ./publish
```
selanjutnya copy ke IIS
